1- First npm command to setup the project
npm init -y 

2- Second npm Command  to setup express
npm i express

3- Third npm Command  to setup nodemon 
npm i -D nodemon

4- Fourth npm Command  to setup mongoose database
 npm i mongoose

5- Fifth npm Command  to setup  dotenv file
 npm i dotenv

6- Sixth npm Command  to setup  jsonwebtoken
npm i jsonwebtoken

7- Seventh npm Command  to setup  bcrpyt for password hassing
npm i bcrpyt

8- Eigth npm Command  to setup  for sending email
npm i nodemailer

9- Ninth npm Command  to setup backend with react
npm i cors
